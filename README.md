## **개요**
> **프로젝트** : 스위치를 활용한 RGB LED 제어하기
> 
> **주제**: 스위치를 누를 때마다 RGB LED에서 색깔이 변화하는 것을 볼 수 있다.
> 
> **주요 기능**
> > - sw 스위치를 누른다.
> > - RGB LED 색이 바뀌게 된다.
